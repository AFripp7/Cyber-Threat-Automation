Cyber Threat Intelligence Automator 

This project automatically collects *public, non-sensitive* cyberthreat intelligence
from open sources such as CVE feeds and security advisories. It summarizes the
information and generates a daily report.

 Features
- Pulls public CVE JSON feeds
- Extracts high-level metadata (ID, vendor, severity)
- Generates Markdown and JSON reports
- Runs automatically using GitHub Actions
